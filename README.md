# 图书馆图书借阅信息管理系统

这是一个用C++和Qt框架开发的图书馆图书借阅信息管理系统，作为数据结构课程设计作业，实现了完整的图书借阅管理功能。

##  主要功能

### 1. 图书管理模块
- 图书的增删改查
- 按标题、作者、类别搜索图书
- 显示图书详细信息（编号、书名、作者、类别、库存状态）
- 批量生成测试数据

### 2. 读者管理模块
- 读者信息的增删改查
- 按ID、姓名、院系搜索读者
- 读者注册信息管理

### 3. 借阅管理模块
- 图书借阅、归还、预定功能
- 支持自定义借阅/归还日期（便于测试）
- 查看图书借阅历史记录
- 查看读者借阅记录
- 逾期记录查询

### 4. 统计分析模块
- 实时统计（总图书数、可借数、已借出数、读者总数等）
- 分类统计（按图书类别）
- 逾期记录显示
- 预定记录查询

### 5. 文件管理
- 新建、打开、保存、另存为数据文件（.lib格式）
- 自动保存/恢复程序设置
- 退出时提示保存

### 6. 时间管理（特色功能）
- 支持设置自定义系统时间（便于测试逾期等功能）
- 可恢复使用实时系统时间

##  项目优点

### 技术实现方面
- **良好的架构设计**
  - 采用MVC模式分离数据、界面和业务逻辑
  - LibraryManager作为核心管理类，封装所有业务逻辑
  - 清晰的类职责划分（Book、Reader、BorrowRecord）

- **完善的数据持久化**
  - 自定义文件格式保存所有数据
  - 支持导入导出
  - 使用QSettings保存程序配置

- **丰富的错误处理**
  - 表单验证（非空检查、逻辑验证）
  - 操作确认对话框
  - 友好的错误提示

- **实用的测试功能**
  - 随机生成测试数据
  - 自定义时间功能便于测试
  - 批量操作支持

### 用户体验方面
- **直观的界面布局**
  - Tab标签页组织功能模块
  - 表格显示数据，支持排序
  - 状态栏实时显示信息

- **便捷的操作流程**
  - 双击表格项自动填充表单
  - 清空搜索条件按钮
  - 一键生成测试数据

- **完善的反馈机制**
  - 操作成功/失败提示
  - 搜索结果统计显示
  - 时间状态清晰标识

##  项目结构

### 核心类说明
```
├── Book (图书类)
│   ├── 属性：ID、标题、作者、类别、库存状态
│   └── 方法：借阅、归还、预定、序列化
│
├── Reader (读者类)
│   ├── 属性：ID、姓名、院系、电话、注册日期
│   └── 方法：序列化
│
├── BorrowRecord (借阅记录类)
│   ├── 属性：读者ID、图书ID、借阅/应还/归还日期
│   └── 方法：序列化、归还操作
│
├── LibraryManager (核心管理类)
│   ├── 管理所有数据和业务逻辑
│   ├── 文件读写、数据持久化
│   ├── 搜索、统计、时间管理
│   └── 信号机制通知UI更新
│
└── MainWindow (主界面类)
    ├── UI布局和控件管理
    ├── 事件处理槽函数
    └── 数据展示和用户交互
```

### 文件组织
```
Library.pro          # Qt项目文件
├── main.cpp         # 程序入口
├── mainwindow.h/cpp # 主窗口界面
├── mainwindow.ui    # 界面设计文件
├── librarymanager.h/cpp # 核心业务逻辑
├── book.h/cpp       # 图书类
├── reader.h/cpp     # 读者类
└── borrowrecord.h/cpp # 借阅记录类
```

##  技术特点

- **面向对象设计**：良好的类封装和继承关系
- **数据序列化**：自定义文件格式保存复杂数据结构
- **信号槽机制**：Qt特有的异步通信机制
- **内存管理**：合理使用智能指针和RAII原则
- **多线程安全**：考虑数据访问的线程安全性

##  数据结构应用

项目中体现了多种数据结构的应用：

- **QMap/QHash**：用于快速查找图书和读者
- **QVector/QList**：存储借阅记录和搜索结果
- **枚举类型**：定义图书状态和类别
- **自定义类**：封装图书、读者、借阅记录等实体
